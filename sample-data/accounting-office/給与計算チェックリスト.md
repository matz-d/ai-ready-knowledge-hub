# 給与計算チェックリスト

## 月次給与計算 共通チェック

### 入力データ
- [ ] 当月の勤怠データ受領 (月末締め翌月5日まで)
- [ ] 残業時間の集計
- [ ] 有給休暇取得日数の確認
- [ ] 通勤手当の変更有無

### 控除関係
- [ ] 健康保険料・厚生年金保険料の標準報酬月額確認
- [ ] 雇用保険料の料率変更チェック
- [ ] 所得税の源泉徴収税額表 最新版適用
- [ ] 住民税の特別徴収額 (5月までに変更分の反映)

### 異動関係
- [ ] 入社・退社者の処理
- [ ] 扶養人数の変更
- [ ] 通勤手段の変更
- [ ] 育児休業・介護休業の有無

### 支給後
- [ ] 給与明細の発行
- [ ] 振込データの作成 (支給日3営業日前まで)
- [ ] 賃金台帳への記載
- [ ] 源泉徴収簿への記載

### 補足
- 法改正対応 (社会保険料率・所得税額表) は年度替わりに最新化が必要
- 顧問先ごとに支給日・締め日が異なる場合があるため、各社の運用ルールを参照すること
